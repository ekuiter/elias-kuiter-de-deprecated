The LED cube software provides three modes of operation: Loop mode, Clap mode and Visual mode.
The three videos on this site show you how these modes work.

Loop mode

By default, Loop mode is active after turning on the LED cube.

In this mode all effects are shown successively in a continuous loop in order that every effect is active for a few seconds.

Internally the effects are called:
fill, scanner, randomize, extend_from_corner, snake, up_down, image, fan

Clap mode

You can activate Clap mode by pressing the mode button at the front of the LED cube once.

In this mode an effect is shown until you clap or make loud noise - then the next effect is shown.

That works as follows: The attached microphone detects the environment volume. If it exceeds a certain threshold (e.g. through clapping), an effect change is triggered. Therefore this mode doesn't work well in a loud environment.

Visual mode

If Clap mode is active, you can switch to Visual mode by pressing the mode button once.

In this mode the LED cube displays the environment volume graphically - so it's also suited for non-MIDI devices like a piano.
