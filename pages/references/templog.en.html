2013 - Concept and design of TempLog for Windows and Android

TempLog is a software for processing temperature data. For using TempLog you need an HCP/WAECO Bluetooth module. Here's a short overview of both of the usages.

TempLog for Windows TempLog for Android

TempLog for Windows stores your temperature data on the internet - always accessible. This works as follows:

1. The Bluetooth module has an integrated temperature sensor. It sends the latest temperature data to the computer.
2. The computer receives the temperature data. A Windows software stores the data on the internet.
3. The internet database receives the data from the PC - it's then stored and graphically displayed (see picture 1).

TempLog for Android shows you the latest temperature data - fast & easy through Bluetooth technology.

For using TempLog for Android, you need a Bluetooth module. Once switched on, it sends recent temperature data to your Android device.

TempLog for Android then displays the temperature as a lettering (see picture 3).
There is a variant by HCP-Technology and one by Dometic WAECO.
