2013 - Entwurf und Design von TempLog für Windows und Android

TempLog ist eine Software, die Temperaturdaten verarbeitet. Dazu benötigt wird ein HCP/WAECO Bluetooth-Modul. Es folgt eine kurze Übersicht über die beiden Verwendungsmöglichkeiten.

TempLog für Windows TempLog für Android

TempLog für Windows speichert Ihre Temperaturdaten im Internet - jederzeit zugänglich. Das funktioniert wie folgt:

1. Das Bluetooth-Modul hat einen integrierten Temperatursensor. Es schickt immer die aktuellsten Temperaturen an den Computer.
2. Der Computer empfängt die Temperaturdaten. Eine Windows-Software schickt die Daten ins Internet.
3. Die Internet-Datenbank nimmt die Daten vom PC entgegen. Die Daten werden gespeichert und grafisch dargestellt (oben Bild 1).

TempLog für Android zeigt Ihnen die aktuellsten Temperaturdaten - schnell & einfach durch Bluetooth-Technologie.

Um TempLog für Android zu nutzen, benötigen Sie ein Bluetooth-Modul. Einmal eingeschaltet, schickt dieses die aktuelle Temperatur an Ihr Android-Gerät.

TempLog für Android zeigt die Höhe der Temperatur dann als Schriftzug an (oben Bild 3).
Es ist jeweils eine Variante von HCP-Technology und Dometic WAECO verfügbar.
